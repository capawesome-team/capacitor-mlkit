# @capacitor-mlkit/smart-reply

⚡️ Capacitor plugin for ML Kit Smart Reply.

## Installation

```bash
npm install @capacitor-mlkit/smart-reply
npx cap sync
```

### Android

#### Variables

This plugin will use the following project variables (defined in your app’s `variables.gradle` file):

- `$mlkitSmartReplyVersion` version of `com.google.mlkit:smart-reply` (default: `17.0.2`)

## Configuration

No configuration required for this plugin.

## Demo

A working example can be found here: [mileszim/capacitor-mlkit-plugin-demo](https://github.com/mileszim/capacitor-mlkit-plugin-demo)

## Usage

```typescript
import { Translation, Language } from '@capacitor-mlkit/smart-reply';

const deleteDownloadedModel = async () => {
  await Translation.deleteDownloadedModel({
    language: Language.English,
  });
};

const downloadModel = async () => {
  await Translation.downloadModel({
    language: Language.English,
  });
};

const getDownloadedModels = async () => {
  const { languages } = await Translation.getDownloadedModels();
  return languages;
};

const translate = async () => {
  const { text } = await Translation.translate({
    text: 'Good morning!',
    sourceLanguage: Language.English,
    targetLanguage: Language.German,
  });
  return text;
};
```

### smartReply(...)

```typescript
smartReply(messages: [TextMessage]) => Promise<{ value: [SmartReplySuggestion]; }>
```

Generate smart replies for the given messages.

Only available on Android and iOS.

| Param          | Type                       |
| -------------- | -------------------------- |
| **`messages`** | <code>[TextMessage]</code> |

**Returns:** <code>Promise&lt;{ value: [SmartReplySuggestion]; }&gt;</code>

**Since:** 0.0.1

--------------------


### Interfaces


#### SmartReplySuggestion

A suggested reply for the given input text.

| Prop       | Type                | Description                                   | Since |
| ---------- | ------------------- | --------------------------------------------- | ----- |
| **`text`** | <code>string</code> | String representation of the suggested reply. | 0.0.1 |


#### TextMessage

A single chat message, to be used as an array element for input to Smart Reply.

| Prop              | Type                                            | Description                                                                     | Since |
| ----------------- | ----------------------------------------------- | ------------------------------------------------------------------------------- | ----- |
| **`text`**        | <code>string</code>                             | A single chat message, to be used as an array element for input to Smart Reply. | 0.0.1 |
| **`timestamp`**   | <code>string \| <a href="#date">Date</a></code> | Timestamp of the chat message.                                                  | 0.0.1 |
| **`userID`**      | <code>string</code>                             | User ID of the message sender.                                                  | 0.0.1 |
| **`isLocalUser`** | <code>boolean</code>                            | Whether this message is from the user that the suggestions are generated for.   | 0.0.1 |
